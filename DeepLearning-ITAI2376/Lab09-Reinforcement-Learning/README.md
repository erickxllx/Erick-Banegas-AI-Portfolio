# Lab 09 — Reinforcement Learning: CartPole with Q-Learning

## Problem Statement
Apply reinforcement learning fundamentals by teaching an agent to balance a pole on a cart using Q-Learning, and compare it against a random baseline agent.

## Approach & Methodology
1. **Part 1 — Random Agent:** Observed a completely untrained agent attempting the CartPole task to establish a failure baseline
2. **Part 2 — Q-Learning Agent:** Implemented and trained a Q-Learning agent that learns from experience through reward-based feedback
3. **Part 3 — Reflection:** Connected RL concepts to broader AI applications and final project design

## Key Concepts
- Reinforcement Learning fundamentals (states, actions, rewards, policies)
- Q-Learning algorithm and Q-table updates
- Exploration vs exploitation trade-off
- Episode-based training and reward accumulation
- OpenAI Gymnasium environment setup

## Results & Evaluation
- Random agent: failed quickly with no learning capability
- Q-Learning agent: learned to balance the pole over many episodes
- Demonstrated the power of reward-based learning vs random behavior

## Requirements
- Python 3.x
- Gymnasium (OpenAI)
- NumPy, Matplotlib
