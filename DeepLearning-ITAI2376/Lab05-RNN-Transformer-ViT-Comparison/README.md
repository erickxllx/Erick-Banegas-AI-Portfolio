# Lab 05 — RNNs vs Transformers vs Vision Transformers

## Problem Statement
Build, train, and compare three major deep learning architectures — RNNs (LSTM & GRU), Transformers, and Vision Transformers (ViT) — to understand their strengths, weaknesses, and ideal use cases.

## Approach & Methodology
1. **Part A — RNN Text Classification:** Built LSTM and GRU models for sequence-based text classification
2. **Part B — Transformer Architecture:** Implemented attention mechanisms and transformer blocks for NLP tasks
3. **Part C — Vision Transformer (ViT):** Applied transformer architecture to image classification by treating image patches as tokens
4. **Comparison:** Evaluated all three architectures on accuracy, training efficiency, and suitability for different data types

## Key Concepts
- Recurrent Neural Networks (LSTM, GRU) and sequential memory
- Self-attention and multi-head attention mechanisms
- Positional encodings for sequence awareness
- Patch embedding for image-to-token conversion
- Architecture trade-offs: inductive bias vs flexibility

## Results & Evaluation
- Compared model performance across text and image tasks
- Analyzed attention patterns and feature representations
- Identified when to use RNNs vs Transformers vs ViTs in practice
