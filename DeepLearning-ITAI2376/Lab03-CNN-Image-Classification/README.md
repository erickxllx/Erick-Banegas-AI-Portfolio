# Lab 03 — CNN Image Classification: Puppy or Bagel? 🐶🥯

## Problem Statement
Build a Convolutional Neural Network to solve a fun binary classification challenge — distinguishing between images of puppies and bagels using TensorFlow/Keras.

## Approach & Methodology
1. Loaded and preprocessed a custom image dataset
2. Built a CNN architecture with convolutional layers, pooling layers, and fully connected layers
3. Applied data augmentation to improve generalization
4. Trained and evaluated the model on the classification task
5. Visualized model predictions and performance metrics

## Key Concepts
- Convolutional layers and feature extraction
- Max pooling and dimensionality reduction
- Data augmentation techniques
- Transfer learning concepts
- Binary classification with CNNs

## Results & Evaluation
- Trained a CNN capable of distinguishing between visually similar classes
- Analyzed model accuracy, loss curves, and prediction confidence
- Explored how CNNs learn hierarchical visual features
