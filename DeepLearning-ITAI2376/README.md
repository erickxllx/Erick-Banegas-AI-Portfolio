## Course Overview

This course covers the theory and practice of deep learning, progressing from foundational neural networks to advanced generative models and reinforcement learning. Each lab involves hands-on implementation using Python, PyTorch, and TensorFlow.

---

## Labs

### Lab 02 — Neural Network Foundations
Build a neural network from scratch using NumPy, then replicate it using PyTorch and TensorFlow. Covers forward propagation, backpropagation, activation functions, and loss computation.

📁 [Lab02-Neural-Network-Foundations](Lab02-Neural-Network-Foundations/)

### Lab 03 — CNN Image Classification (Puppy vs Bagel)
Implement a Convolutional Neural Network for binary image classification. Explores convolutional layers, pooling, data augmentation, and transfer learning with a fun dataset.

📁 [Lab03-CNN-Image-Classification](Lab03-CNN-Image-Classification/)

### Lab 05 — RNN vs Transformer vs Vision Transformer
Build, train, and compare three major architectures: LSTM/GRU for text classification, Transformers with attention mechanisms, and Vision Transformers (ViT) for image tasks.

📁 [Lab05-RNN-Transformer-ViT-Comparison](Lab05-RNN-Transformer-ViT-Comparison/)

### Lab 08 — Diffusion Model for Image Generation
Implement a diffusion-based generative model from scratch. Build a U-Net with time embeddings, implement forward/reverse diffusion processes, and evaluate with CLIP.

📁 [Lab08-Diffusion-Model](Lab08-Diffusion-Model/)

### Lab 09 — Reinforcement Learning (CartPole)
Implement Q-Learning to teach an agent to balance a pole. Compare random vs trained agents and explore reward-based training strategies.

📁 [Lab09-Reinforcement-Learning](Lab09-Reinforcement-Learning/)
