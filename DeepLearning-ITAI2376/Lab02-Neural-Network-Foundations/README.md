# Lab 02 — Neural Network Foundations: From Scratch to Frameworks

## Problem Statement
Understand how neural networks work at a fundamental level by building one from scratch, then compare implementations across PyTorch and TensorFlow.

## Approach & Methodology
1. Built a neural network using only Python and NumPy — implementing forward propagation, backpropagation, and gradient descent manually
2. Compared PyTorch and TensorFlow as deep learning frameworks
3. Prepared and preprocessed data for training
4. Trained and evaluated models on classification tasks

## Key Concepts
- Perceptrons and multi-layer networks
- Activation functions (ReLU, Sigmoid, Softmax)
- Loss functions and optimization
- Manual vs framework-based backpropagation
- PyTorch dynamic graphs vs TensorFlow static graphs

## Results & Evaluation
- Successfully implemented a working neural network from scratch
- Compared training behavior and API design between PyTorch and TensorFlow
- Demonstrated understanding of the core math behind deep learning

## Requirements
- Python 3.x
- PyTorch
- TensorFlow
- NumPy, Matplotlib
