## Featured Project: Diffusion Model for Image Generation

Built a diffusion-based generative model from scratch using PyTorch, capable of generating realistic handwritten digit images from pure noise.

**Highlights:**
- Implemented U-Net architecture with sinusoidal time embeddings and skip connections
- Designed forward (noise addition) and reverse (denoising) diffusion processes
- Trained with MSE loss, gradient clipping, and early stopping
- Evaluated generated image quality using CLIP (Contrastive Language-Image Pre-training)
